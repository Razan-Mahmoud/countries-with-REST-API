# Rest Countries with Theme Toggle

This is a Frontend Mentor challenge. Built with Next.js 14, Tailwind CSS, Schadcn, React, and TypeScript. The website ensures a smooth, fast, and elegant browsing experience across all devices.

Solution URL: [REST Countries API Exercise](https://rest-countries-api-mu-five.vercel.app/)

## Table of contents
   - [Features](#features)
  - [Web Pages](#web-pages)
  - [Links](#links)
  - [Tech Stack](#teck-stack)

### Features
- Modern UI/UX – Fully responsive and built with Tailwind CSS for beautiful design.
- Light/Dark Mode – Smooth toggle for better accessibility and user comfort.
- Search & Filters – Improved search with filters by region and a search input.
- Type Safety – Entire app written in TypeScript.

### Web Pages
- Home – Get all countries, with search input, region filter and dark theme toggle.
- Country Card – Get Information about a specific country.

### Links
It is a solution to the [REST Countries API with color theme switcher challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/rest-countries-api-with-color-theme-switcher-5cacc469fec04111f7b848ca).

### Tech Stack
- Framework - Next.js 14 With TypeScript
- Styling - Tailwind CSS, tailwindcss-animate
- UI Components - Shadcn/UI
- Theme Support - next-themes
- Linting & Formatting - ESLint, Prettier, prettier-plugin-tailwindcss Next.JS
- Libraries: Many thanks to [@yusifaliyevpro](https://github.com/yusifaliyevpro/countries) for building and sharing their libraries. They were beneficial and smart.
